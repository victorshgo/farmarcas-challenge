# CRUD - Farmarcas

```bash

# Abra o terminal/CMD e clone este repositório
$ git clone https://github.com/victorshgo/farmarcas-challenge.git

# Acesse a pasta do projeto
$ cd farmarcas-challenge

# Acesse a pasta da API
$ cd api

# Instale as dependências
$ yarn

# Execute a aplicação
$ yarn dev

# Em outra instância do terminal/CMD, acesse a pasta do projeto novamente
$ cd farmarcas-challenge

# Instale as dependências
$ yarn

# Execute a aplicação
$ yarn start

# Deixe os dois terminais rodando em paralelo.

```
